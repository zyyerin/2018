–––  installation

The viewer’s location is tracked by a depth sensor, and visuals are generated and projected on the ground, merging into the viewer’s own shadow. The visuals are programmed in Processing, using a physics engine library.

proposal

This is a half-semester long project. Final installation is shown in the interface show at the end of the semester. The work sheet helps me to plan ahead and keep orginized.

experiments

The project invovles using physics libray, kinnect sensor, and working with computer vision, which were new to me. I make small sketches to test the algrithums and eventually managed put everything toghether.
